Reporting Demonstrations

The following examples demonstrate how third-party developers can write web applications to query data from the dashboard and display reports:

Task List
The list of tasks in this project that have been assigned to individuals. Demonstrates an HQL query against the TaskStatusFact table.
Time Log
The time log for this project. Demonstrates an HQL query against the TimeLogFact table, as well as the use of mapped process phases.
Defect Log
The defect log for this project. Demonstrates an HQL query against the DefectLogFact table.
Plan Summary
A partial plan summary for this project. Demonstrates the retrieval of precalculated metrics from the data repository.
Component Data Dump
A CSV table of project data by component, in a CSV format that can be opened in Excel. Demonstrates the execution of multiple HQL queries from Java servlet logic (rather than directly from a JSP).

Useful Tools

The following links can be helpful for report developers:

HQL Query Tester
This page will allow you to enter and run an HQL query. It can be a helpful way to test and debug problems with HQL syntax, content, etc.
Reload WAR Files
If you are rebuilding WAR files and republishing them to a running dashboard, you can click the link above to reload the WAR files that have changed. (Note this link will not detect newly added WAR files; to accomplish that you must restart the dashboard.)
